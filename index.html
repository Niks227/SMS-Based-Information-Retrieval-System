Project Description

Usually we don't know where our classes our and if in case we know we don't have the study material ready with us. This Project was created so that students never miss their class again and get the desired material with just a mail.

Features

  • After subscribing , the person get a short message service fifteen minutes before the actual class timing with all the relevant details.
  • Easy access of study material, by simply sending a mail with relevant information. # Technology/Tools
  • Xamp
  • php
  • mySql
  • Cron Jobs
